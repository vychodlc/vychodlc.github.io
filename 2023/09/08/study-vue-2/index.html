Vue 组件通信方式

  • 通信的类型
    • 父子组件通信
    • 兄弟组件通信
    • 祖孙、后代通信
    • 非关系组件的通信
  • 通信的方式（8种）
    1. props
    2. $emit
    3. ref
    4. EventBus
    5. parent root
    6. attrs listeners
    7. provide inject
    8. vuex 等状态管理库

  • 详细描述

    • props 通信：父向子传递数据

    • $emit 通信：子组件触发父组件

    • ref 通信：

    • EventBus：兄弟组件传值，总线机制

    • parent root 通信：适用于有共同祖先的组件

    • attrs listeners 通信：祖先传递给子孙

    • provide inject 通信

    • 其他：vuex 状态管理，适合组件之间更加复杂的数据传输
